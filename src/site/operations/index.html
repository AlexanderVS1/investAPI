    OperationsService

    Сервис предназначен для получения:
    1. списка операций по счёту;
    2. портфеля по счёту;
    3. позиций ценных бумаг на счёте;
    4. доступного остатка для вывода средств.

    Методы сервиса

    GetOperations

    Метод получения списка операций по счёту

    • Тело запроса — OperationsRequest

    • Тело ответа — OperationsResponse

    GetPortfolio

    Метод получения портфеля по счёту

    • Тело запроса — PortfolioRequest

    • Тело ответа — PortfolioResponse

    GetPositions

    Метод получения списка позиций по счёту

    • Тело запроса — PositionsRequest

    • Тело ответа — PositionsResponse

    GetWithdrawLimits

    Метод получения доступного остатка для вывода средств

    • Тело запроса — WithdrawLimitsRequest

    • Тело ответа — WithdrawLimitsResponse

    Сообщения методов

    OperationsRequest

    Field Type Description
    account_id string Идентификатор счёта клиента
    from google.protobuf.Timestamp Начало периода (по UTC)
    to google.protobuf.Timestamp Окончание периода (по UTC)
    state OperationState Статус запрашиваемых операций
    figi string Figi-идентификатор инструмента для фильтрации
    limit int32 Количество запрашиваемых операций (по умолчанию 100)
    marker int32 Положение курсора (по умолчанию 0)

    OperationsResponse

    Field Type Description
    total int32 Общее количество операций по счёту
    marker int32 Текущее положение курсора
    operations Массив объектов Operation Массив операций

    Operation

    Данные по операции

    Field Type Description
    id string Идентификатор операции
    parent_operation_id string Идентификатор родительской операции
    currency string Валюта операции
    payment MoneyValue Сумма операции
    price MoneyValue Цена операции
    state OperationState Статус операции
    quantity int64 Количество лотов инструмента
    quantity_executed int64 Исполнено лотов
    figi string Figi-идентификатор инструмента, связанного с операцией
    instrument_type string Тип инструмента. Возможные значения:
    bond — облигация;
    stock — акция;
    currency — валюта;
    etf — фонд;
    futures — фьючерс.
    date google.protobuf.Timestamp Дата и время операции в формате часовом поясе UTC
    type string Тип операции

    PortfolioRequest

    Field Type Description
    account_id string Идентификатор счёта пользователя

    PortfolioResponse

    Field Type Description
    total_amount_bonds MoneyValue Текущая стоимость всех облигаций в рублях
    total_amount_currencies MoneyValue Текущая стоимость всех валютных позиций в рублях
    total_amount_etf MoneyValue Текущая стоимость всех ETF в рублях
    total_amount_stocks MoneyValue Текущая стоимость всех акций в рублях
    total_amount_portfolio MoneyValue Текущая стоимость портфеля в рублях
    portfolio_yield MoneyValue Абсолютная доходность портфеля за все время с учетом валютной переоценки
    portfolio_yield_relative float Относительная доходность портфеля на все время с учетом валютной переоценки
    portfolio_daily_yield MoneyValue Абсолютная дневная доходность портфеля с учетом валютной переоценки
    portfolio_daily_yield_relative float Относительная дневная доходность портфеля с учетом валютной переоценки
    items Массив объектов PortfolioItem Список позиций портфеля

    PositionsRequest

    Field Type Description
    account_id string Идентификатор счёта пользователя

    PositionsResponse

    Field Type Description
    money Массив объектов MoneyValue Массив валютных позиций портфеля
    blocked Массив объектов MoneyValue Массив заблокированных валютных позиций портфеля
    securities Массив объектов PositionsSecurities Список ценно-бумажных позиций портфеля
    limits_loading_in_progress bool Признак идущей в данный момент выгрузки лимитов

    WithdrawLimitsRequest

    Field Type Description
    account_id string Идентификатор счёта пользователя

    WithdrawLimitsResponse

    Field Type Description
    money Массив объектов MoneyValue Массив валютных позиций портфеля
    blocked Массив объектов MoneyValue Массив заблокированных валютных позиций портфеля

    PortfolioItem

    Field Type Description
    ticker string Тикер
    figi string Figi-идентификатор инструмента
    instrument_type string Тип инструмента. Возможные значения:
    bond — облигация;
    stock — акция;
    currency — валюта;
    etf — фонд;
    futures — фьючерс.
    quantity int64 Количество лотов
    currency string Валюта расчётов по инструменту
    price MoneyValue Текущая цена инструмента
    avg_cost_price MoneyValue Средняя цена покупки позиции с учетом накопленного купонного дохода
    avg_cost_price_no_acc MoneyValue Средняя цена покупки позиции без учета накопленного купонного дохода
    current_amount MoneyValue Текущая стоимость позиции
    accruedint MoneyValue Накопленный купонный доход
    yield MoneyValue Абсолютная доходность за все время с учетом валютной переоценки
    yield_relative float Относительная доходность на все время с учетом валютной переоценки
    daily_yield MoneyValue Абсолютная дневная доходность с учетом валютной переоценки
    daily_yield_relative float Относительная дневная доходность с учетом валютной переоценки
    coupons_acquired_amt MoneyValue Сумма операций по выплате купонов
    coupons_acquired_qty int32 Количество операций по выплате купонов
    current_var_margin float Вариационная маржа (параметр возвращается только для фьючерсов)

    figi string Figi-идентификатор бумаги
    blocked int64 Заблокировано
    balance int64 Текущий баланс

    Name Number Description
    OPERATION_STATE_UNSPECIFIED 0 Статус операции не определён
    OPERATION_STATE_EXECUTED 1 Исполнена
    OPERATION_STATE_CANCELED 2 Отменена

    Field Type Description
    currency string Строковый ISO-код валюты
    units int64 Целая часть суммы, может быть отрицательным числом
    nano int32 Дробная часть суммы, может быть отрицательным числом
